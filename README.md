# ANC_en_ducto
Control Activo de Ruido en ducto.
Repositorio del Proyecto Final para el Examen de la Asignatura optativa "Acústica y Electroacústica", formalmente llamada "Complementos de Electrónica Analógica I". 
### Autores: 
- Oieni Orozco,  Valentina
- Suárez, Alejo

Proyecto desarrollado en Área de Acústica del Instituto de Automática (INAUT) de la Universidad Nacional de San Juan, Argentina (UNSJ).
